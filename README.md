# 오늘의 운세 🔮

이름과 생년월일을 입력하면 오늘의 운세를 알려주는 웹 애플리케이션입니다.

## 기능

- 이름과 생년월일 입력
- 생년월일 기반 오늘의 운세 생성
- 아름다운 UI/UX

## 기술 스택

- React 18
- Vite
- CSS3

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```

## Vercel 배포

1. GitHub에 프로젝트를 푸시합니다
2. [Vercel](https://vercel.com)에 로그인합니다
3. "New Project"를 클릭하고 GitHub 저장소를 선택합니다
4. Vercel이 자동으로 설정을 감지하고 배포합니다

또는 Vercel CLI를 사용하여 배포할 수 있습니다:

```bash
npm i -g vercel
vercel
```
